# 🎮 Number Guessing Game

A fun and interactive command-line number guessing game where players try to guess a randomly generated number between 1 and 100. Test your luck and intuition across different difficulty levels!

## 📋 Features

- **Three Difficulty Levels**: 
  - Easy (10 chances)
  - Medium (5 chances)  
  - Hard (3 chances)

- **Multiple Input Methods**: Choose difficulty by name (easy/medium/hard) or number (1/2/3)

- **Smart Validation**: 
  - Ensures guesses are valid numbers between 1-100
  - Doesn't count invalid attempts against your chances

- **Real-time Feedback**: 
  - Tells you if your guess is too high or too low
  - Shows number of attempts used

- **Timer Feature**: Tracks how long it takes you to guess the number

- **Replayability**: Option to play multiple rounds without restarting

- **User-friendly Interface**: Clear instructions and formatted output for better readability

## 🚀 Installation

1. **Clone the repository** (or download the source code):
2. **install dependencies**: npm install


##  🎯 How to Play
1. **Start the game**: npm start
2. **Choose difficulty level when prompted**:
    Type easy or 1 for Easy mode (10 chances)

    Type medium or 2 for Medium mode (5 chances)

    Type hard or 3 for Hard mode (3 chances)
